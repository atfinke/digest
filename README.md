# digest

Generates a daily gmail unread digest. Perfect for abandoned university accounts. Install using `pip install -r requirements.txt`.

Email format: 
```
Subject: NU {m}/{d} Digest ({#} Unread)

---

Inbox ({#}):
- {unread email subject #1)
- ...

{Filter Label Subject #1}
- {unread email subject #1)
- ...

...

/eom

```
